# Rupam Kumari - Portfolio

A modern, responsive portfolio website showcasing projects, certifications, achievements, and learnings.

## Features

- Responsive design
- GitHub projects integration
- Smooth scrolling navigation
- Contact form
- Modern CSS animations

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Font Awesome icons

## Getting Started

1. Clone the repository
2. Open `index.html` in your browser

## Customization

- Update personal information in `index.html`
- Replace `images/profile-placeholder.svg` with your actual profile photo as `images/profile.jpg` and update the `src` path in `index.html` if necessary
- Modify styles in `css/style.css`
- Add interactivity in `js/script.js`
